# Nesne_-dev_Kuantum_Kaos_Yönetimi
README.md

Bu proje, aynı program mantığını üç farklı dilde (C#, Python, JavaScript ve Java) yazmayı amaçlayan bir çalışma örneğidir. Programın temel amacı; kullanıcıya bir menü sunmak, seçimlere göre nesne eklemek, listelemek, analiz yapmak ve hata durumlarını özel mesajlarla göstermek üzerine kuruludur.

 Projede Neler Var?

- **C# sürümü:** Konsolda çalışan menü yapısı, özel exception sınıfları ve kullanıcı girişleri.
- **Python sürümü:**C# ile aynı ilerleyip sadece .py ile derlenmiştir.
- **JavaScript sürümü (Node.js):** C#’taki yapının birebir mantığı korunarak terminal tabanlı şekilde çalışır.
- **Java sürümü:** JDK ile derlenip çalıştırılabilen menü sistemi ve hata yönetimi.

 Projenin Amacı

Aynı uygulamayı farklı programlama dillerinde yazarak:
- diller arası sözdizimi farklarını görmek,
- giriş–çıkış işlemlerinin nasıl değiştiğini öğrenmek,
- temel program akışını her dilde kurmayı deneyimlemek.

 Çalıştırma

# C#
dotnet run


# JavaScript
node program.js

#Python ve java ise direkt çalıştırma tuşu ile çalışabilir.
